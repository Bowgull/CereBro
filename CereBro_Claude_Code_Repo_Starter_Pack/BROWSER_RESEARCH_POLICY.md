# CereBro V1 — Browser Research Policy

## Core Rule

Browser research must be source-grounded, logged, and permissioned.

## Source Trust Levels

- official
- primary
- high
- medium
- low
- unknown

## Browser Use Rules

Browser tools are disabled by default.

Enable only with user approval.

Private/logged-in browsing requires explicit approval.

## Extraction Order

1. User-provided text
2. Public URL metadata
3. Static extraction
4. Browser-assisted extraction
5. Screenshot
6. Optional Crawl4AI adapter
7. Optional advanced extraction with approval

## GitHub Review Format

Every GitHub repo review should include:

- what it is
- why it matters
- license concern
- CereBro decision
- module impact
- what to steal
- what not to steal
- next action

## Reddit Rule

Use Reddit for lived-experience signals and pattern discovery.

Do not treat a single Reddit comment as truth.
