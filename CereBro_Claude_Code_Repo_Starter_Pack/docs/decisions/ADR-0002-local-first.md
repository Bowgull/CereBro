# ADR-0002 — Local-First Architecture

## Status

Accepted.

## Decision

CereBro remains local-first.

## Reason

The user wants control, low cost, local memory, and no forced hosting.

## Consequences

Use:

- local Next.js app
- SQLite
- local files
- Obsidian
- Chroma
- Ollama
- optional Notion
- optional external models with approval
