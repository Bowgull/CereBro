# CereBro V1 — Acceptance Criteria

V1 is accepted only when all required criteria pass.

## Required

- App launches locally.
- First-run onboarding works.
- Settings persist.
- Project Spaces work.
- Tasks persist.
- Sessions persist.
- Harness routes tasks.
- Agent registry loads.
- Skills load from files.
- Source Library works.
- Output Library works.
- Memory proposals work.
- Oak validation works.
- Obsidian write works.
- Chroma integration works or is safely stubbed with clear status.
- Notion is optional and safe.
- Tony can create Claude Code handoff.
- Gojo can create UI spec.
- Piccolo can run safe cleanup scan.
- Model Router exists.
- Permissions are enforced.
- Tool calls are logged.
- Errors are structured.
- Tests exist and run.
- Raven Reviews remains sealed/disabled unless explicitly enabled.
- Declyne remains removed from V1.

## Must Not Happen

- No required external SSD.
- No required Notion.
- No required external model.
- No required browser automation.
- No destructive cleanup silently.
- No agents bypass harness.
- No important output bypasses Oak.
- No direct memory writes by agents.
