Blueprint decisions

The castle is now a fixed planning grid. PixelLab room prompts should use these coordinates and keep the same side-cutaway camera.

Grid

64 cols x 34 rows. One planning tile maps to one Phaser logical tile. Pixel art can render at 16 px or 32 px as long as the room anchors stay fixed.

Paths

Aang starts at ground. The main path climbs to Cortana, crosses the command lane, and reaches Gojo through the upper stair. The undercroft gets a smaller maintenance stair.

PixelLab rule

Generate rooms inside this template. Fixed dimensions. Same camera. Door anchors visible. Walking lane clear. Back wall props only. No standalone fantasy painting.
